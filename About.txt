SLIDE 8 вҖ“ FEATURE 2: LIVE LOCATION SHARING
================================================================================

LIVE LOCATION SHARING вҖ” HOW IT WORKS

USER SIDE:
  вҖў User toggles "Share Location" on the Dashboard
  вҖў Browser's Geolocation API (watchPosition) tracks GPS in real time
  вҖў Coordinates are sent to the server and saved in the Locations collection
  вҖў A unique public URL is generated: /share/:userId

RECIPIENT SIDE (no app needed):
  вҖў Anyone with the link opens it in any browser
  вҖў The Share page fetches the latest location every 15 seconds
  вҖў Location is displayed on an interactive Mapbox GL map
  вҖў Shows: latitude, longitude, live status indicator, last updated time

KEY CODE (useGeolocation hook):
  watchId.current = navigator.geolocation.watchPosition(
    (p) => {
      const coords = { lat: p.coords.latitude, lng: p.coords.longitude };
      setPos(coords);
      onUpdate?.(coords);  // pushes to server
    },
    { enableHighAccuracy: true, maximumAge: 10000 }
  );

AUTO-REFRESH (Share page):
  useEffect(() => {
    load();
    const t = setInterval(load, 15000);  // refresh every 15 seconds
    return () => clearInterval(t);
  }, [userId]);

SECURITY:
  вҖў Share URL is public but contains only location data
  вҖў No personal information (name, phone, email) is exposed
  вҖў End-to-end encrypted label displayed to reassure recipients

SLIDE 9 вҖ“ FEATURE 3: SAFE ZONE DISCOVERY
================================================================================

SAFE ZONES вҖ” HOW IT WORKS

PURPOSE:
  Help users quickly find the nearest safe locations during an emergency.

PLACE TYPES SUPPORTED:
  рҹ”ө Police Stations
  рҹҹў Hospitals
  рҹҹЎ Bus Stops

FLOW:
  1. User opens the Safe Zones page
  2. Browser captures current GPS coordinates
  3. User taps "Find Safe Places"
  4. Server queries Google Places API for each type within 2 km radius
  5. Results are sorted by distance (Haversine formula)
  6. Displayed on Mapbox GL map + scrollable list with:
     вҶ’ Place name and type badge
     вҶ’ Distance (meters or km)
     вҶ’ Google rating (stars)
     вҶ’ "Nearest" badge on closest result
     вҶ’ Direct navigation link to Google Maps

HAVERSINE DISTANCE FORMULA:
  function getDistance(lat1, lon1, lat2, lon2) {
    const R = 6371;  // Earth radius in km
    const dLat = (lat2 - lat1) * Math.PI / 180;
    const dLon = (lon2 - lon1) * Math.PI / 180;
    const a = Math.sin(dLat/2)**2 +
              Math.cos(lat1*Math.PI/180) * Math.cos(lat2*Math.PI/180) *
              Math.sin(dLon/2)**2;
    return R * 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
  }

FILTER OPTIONS:
  All | Police | Hospital | Bus Stop

  вҖў Password hashed with bcrypt (salt factor: 10)
  вҖў Duplicate email check

  вҖў Input: email, password
  вҖў bcrypt.compare() validates password
  вҖў Returns: JWT token + user object

  const token = req.headers.authorization?.slice(7);
  const payload = jwt.verify(token, process.env.JWT_SECRET);
  req.user = { id: payload.id, email: payload.email };

CLIENT-SIDE:
  вҖў Token stored in localStorage as 'rakshika_token'
