# Rashika - Women Safety SOS (MERN-ready)

Premium redesign of a women safety SOS web app with a modern glassmorphism aesthetic, responsive dashboard, animated SOS interaction, and emergency workflow modules.

## Stack
- **Frontend:** React + Vite + Framer Motion + Lucide icons
- **Backend:** Node.js + Express

## Features
- Premium sticky navbar with blur/glass effect, theme toggle, profile CTA
- Animated SOS trigger with confirmation modal and alert/safe status indicator
- Emergency contacts management (add/edit/delete)
- Live location map placeholder and device status indicators (battery/network/signal)
- Emergency history timeline
- Panic mode, fake call customization, real-time tracking, and notification settings
- Dashboard cards for alerts, location recency, and tracking state

## Run locally
```bash
npm install
npm --prefix client install
npm --prefix server install
npm run client:dev
npm run server:dev
```

The frontend runs on default Vite port and backend on `http://localhost:5000`.
