# Rashika Safety SOS (MERN)

A premium, production-style Women Safety SOS web application with a modern glassmorphism UI, smooth animations, and modular MERN architecture.

## Stack

- **Frontend:** React + Vite + CSS modules style architecture
- **Backend:** Node.js + Express
- **Database (ready):** MongoDB hooks prepared in service layer

## Features

- Premium fixed glass navbar with theme toggle + profile menu
- Animated SOS center section with pulse glow and confirmation modal
- Live safety status + siren overlay when alert is active
- Dashboard stats cards with motion and hover transitions
- Emergency contacts management (Add / Edit / Delete)
- Location map placeholder, location tracking toggle, panic mode toggle
- Battery and network status indicators
- Fake call customizer (name + ringtone)
- Emergency history log with timestamps
- Settings panel for auto-SOS hold timer, notifications, theme
- Fully responsive mobile-first UI with modern typography (Poppins + Inter)

## Run locally

```bash
npm install
npm run dev
```

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:5000`

## Scripts

- `npm run dev` – run frontend and backend concurrently
- `npm run dev:client` – run React app
- `npm run dev:server` – run API server
- `npm run build` – build frontend
